# 🏦 Predicting Bank Churn using Machine Learning
**Author:** Ishan Kapadia • [LinkedIn](https://www.linkedin.com/in/imkapadia) 

This project uses machine learning to predict whether a customer will churn (leave the bank) based on their profile and banking history.

---

## 🔍 Project Highlights

- 🔎 Exploratory Data Analysis (EDA)
- ⚙️ Preprocessing: encoding, scaling, train-test split
- 🤖 Models: Logistic Regression & Random Forest
- 📊 Evaluation: Classification report, confusion matrix, ROC-AUC
- 👤 Interactive customer prediction: Enter new customer details and get churn probability

---

## 📁 Dataset

The dataset `Bank Customer Churn Prediction.csv` contains:
- 10,000 rows × 12 columns
- Target variable: `churn` (1 = Yes, 0 = No)

---

## 🧠 Models & Evaluation

- **Logistic Regression**: Good baseline model
- **Random Forest**: Better performance and feature insights
- **Metrics Used**:
  - Precision, Recall, F1-Score
  - ROC-AUC
  - Confusion Matrix
  - Feature Importance Plot
